⚽ Política de Privacidade — Futdle

Última atualização: abril de 2026

O Futdle é um jogo de adivinhação de times de futebol desenvolvido para Android. Este aplicativo é operado por Matheus Nascimento. Esta política explica quais dados coletamos, como os usamos e como os protegemos.

1. Dados coletados

  • Identificador anônimo: ao abrir o app pela primeira vez, um ID de usuário anônimo é gerado automaticamente para salvar seu progresso.
  • Apelido (nickname): nome escolhido para exibição em rankings públicos.
  • E-mail e senha (opcional): caso você crie uma conta, seus dados são armazenados de forma segura via Supabase.
  • Progresso do jogo: tentativas, tempo, desafios jogados e energia.
  • Dados de dispositivo e publicidade: identificadores como Advertising ID podem ser coletados para exibição e personalização de anúncios.

2. Como usamos os dados

  • Exibir rankings com apelidos dos jogadores
  • Salvar e sincronizar progresso
  • Garantir funcionamento do sistema de jogo
  • Exibir anúncios e melhorar a experiência do usuário

3. Compartilhamento de dados

Seus dados não são vendidos. Utilizamos serviços de terceiros:

  • Supabase — banco de dados e autenticação. Política de Privacidade
  • Google AdMob — anúncios. Pode coletar identificadores e dados de uso conforme política do Google.

4. Retenção e exclusão de dados

Os dados são armazenados enquanto você utiliza o aplicativo. Você pode solicitar a exclusão a qualquer momento pelo e-mail de contato. Usuários anônimos podem perder acesso aos dados ao desinstalar o app.

5. Segurança

Utilizamos criptografia (HTTPS/TLS). Senhas são protegidas com hash seguro via Supabase Auth.

6. Crianças

O aplicativo não é direcionado a menores de 13 anos e não coleta intencionalmente dados dessas crianças.

7. Transferência internacional de dados

Seus dados podem ser processados fora do Brasil, de acordo com os serviços utilizados.

8. Base legal

  • Execução de contrato (funcionamento do app)
  • Legítimo interesse (melhoria da experiência)
  • Consentimento (quando aplicável, como anúncios personalizados)

9. Seus direitos

  • Acessar seus dados
  • Corrigir dados incompletos ou incorretos
  • Solicitar exclusão
  • Revogar consentimento

10. Alterações nesta política

Podemos atualizar esta política. Mudanças relevantes serão comunicadas no aplicativo.
